# 音频文件说明

这个文件夹用于存放游戏的音频文件。

## 支持的格式
- MP3格式的音乐文件

## 推荐的文件名
- `game_music.mp3` - 游戏背景音乐
- `background_music.mp3` - 背景音乐
- `snake_game_music.mp3` - 贪吃蛇游戏音乐

## 使用说明
1. 将MP3格式的音乐文件放入此文件夹
2. 游戏会自动检测并播放第一个找到的MP3文件
3. 音乐会在游戏开始时自动播放，游戏暂停时会暂停，游戏结束时停止

## 注意事项
- 确保音频文件格式为MP3
- 建议使用较小的文件大小以优化游戏性能
- 音频文件应该是循环播放友好的（开头和结尾能够平滑过渡）

## 音频控制
- 游戏暂停时音乐自动暂停
- 游戏恢复时音乐自动恢复
- 返回主菜单时音乐停止
- 重新开始游戏时音乐重新开始播放
